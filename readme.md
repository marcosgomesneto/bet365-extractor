# [Getting started](https://github.com/microsoft/TypeScript-Node-Starter#getting-started)

### Clone Repository

```
git clone --depth=1 https://github.com/marcosgomesneto/bet365-extractor.git <project_name>
```

### Install Dependencies

- NPM

```
cd <project_name>
npm install
```

- YARN

```
cd <project_name>
yarn
```

### Execute Project

- NPM

```
npm run start:dev
```

- YARN

```
yarn start:dev
```
